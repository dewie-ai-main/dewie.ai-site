what dewie ships out of the box

⚡
Auto-enrichment at ingest
Every document gets topics, entities, keywords, Q&A pairs, reading level, and a quality score — automatically. No schema design, no pipeline config.
🕸
Graph traversal as agent tools
expand, bridge, intersect — named operations agents call natively. Answer questions that require connecting documents no flat search would link.
🧭
ResultConfidence routing
Every query returns a complexity signal: lookup, ambiguous, or distributed — plus a suggested next action so agents know how to retrieve, not just what to retrieve.
🚨
Gap signal
Fires before generation when the corpus doesn't have the answer. Your agent says "I don't know" instead of making something up. The only pre-retrieval coverage signal in the market.

gap_signal

The failure mode no one is solving

Every retrieval API returns a similarity score when it can't find a good answer. Agents can't threshold on raw scores — in a large corpus, irrelevant documents score just as high as relevant ones. So agents confidently hallucinate. Dewie fires a structured signal before retrieval when the corpus genuinely doesn't cover the query. It's not a reranking problem. It's a coverage problem.

Without gap signal
# Agent gets back 10 results
# All score ~0.016 (irrelevant)
# Agent doesn't know to stop

>> "The 2024 Icelandic volcanic
survey found that..."
# hallucination. confident.
With Dewie gap signal
result_confidence:
  gap_signal: "10 results returned but
    none address this query.
    No coverage for: volcanic,
    eruption, survey."
  suggested_action: "none"
# agent stops. user informed.
